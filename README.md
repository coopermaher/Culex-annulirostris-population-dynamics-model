Parameters_data_points.csv is used in Cx annulirostris temperature-depdenent parameters.ipynb as the data file with all the relevant data points of _Culex annulirostris_ parameters used for fitting.
Russell_1986_data_extraction.csv is the digitally extracted _Cx. annulirostris_ numbers that are used to compare the model accuracy in Model 1.ipynb, Model 2.ipynb, Model 3.1.ipynb, Model 3.2.ipynb, Model 4.1.ipynb and Model 4.2.ipynb.
Kanyapella_previous_30_years.csv is the climate data file used for fitting thresholds for the rainfall and water balance values in Water indices thresholds.ipynb
Kanyapella_-36.15_144.85.csv is the climate data file used for running the model in Model 1.ipynb, Model 2.ipynb, Model 3.1.ipynb, Model 3.2.ipynb, Model 4.1.ipynb and Model 4.2.ipynb and plotting the climate values for Climate data.ipynb.
Climate data.ipynb is used for plotting the climate values in the sample site over the study period.
Cx annulirostris temperature-depdenent parameters.ipynb is used for fitting _Cx. annulirostris_ parameters and plotting.
Model 1.ipynb, Model 2.ipynb, Model 3.1.ipynb, Model 3.2.ipynb, Model 4.1.ipynb and Model 4.2.ipynb were all used to run the relevant model varitions. These include figures and metric outputs.
Water indices thresholds.ipynb is used to fit the thresholds for rainfall and water balance values and plot the relevant figure of the classification of periods in the sample site.
